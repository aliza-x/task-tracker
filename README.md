# 📋 Task Tracker — MERN Stack

A full-stack Task Tracker web application built with MongoDB, Express.js, React, and Node.js.

## Features
- Create, View, Update & Delete tasks (CRUD)
- Filter by status and priority
- Sort by date / priority
- Responsive UI
- REST API with Express
- MongoDB Atlas integration

## Tech Stack
- **Frontend**: React.js — [Live Demo](https://task-tracker-tau-pearl-71.vercel.app)
- **Backend**: Node.js + Express.js
- **Database**: MongoDB Atlas

## Setup
Clone the repo, add your `.env` files and run `npm install && npm start` in both `backend/` and `frontend/`.
